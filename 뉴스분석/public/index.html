Gemini API Key 설정

이 서비스는 Google Gemini API와 Search Grounding을 활용합니다.
로컬 `.env` 파일에 키를 설정하지 않았다면 아래에 직접 입력할 수 있습니다.

키는 브라우저 로컬 스토리지에만 안전하게 임시 저장됩니다.

⚠️ Gemini API Key가 설정되지 않았습니다. 정상 작동을 위해 우측 상단 ⚙️ 설정을 클릭해 키를 등록해 주세요.
